Notes 2: There are several examples of events, including onload and onclick. The abbreviation DOM stands for document object model, and it refers to the different objects, or tags, in an HTML file that a script can alter.

    When you click a button on a web page, that event can trigger code that you've written (the function) to execute. The browser reads the HTML of that page and executes specific functions depending on what tag was clicked. That tag is part of the document object model, so it will be found in the HTML of the web page.

    The DOM is a tree structure, and each node in the tree has a parent and children. The parent is the node that contains the child. The child is the node that is contained by the parent. The root node is the node that is not contained by any other node. The root node is the top of the tree.
