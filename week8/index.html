    Notes:

    Transform: The CSS3 Transforms Module allows you to translate, rotate, scale and skew an HTML element. Transform allows you to change how an element moves or looks, like making it smaller or larger. It is a set of properties that link together the visual and motion representation of the element. And it gives your animated elements three dimensional directions.

    Rotate: The CSS3 Rotate Module allows you to rotate an HTML element. With CSS3 Rotate, you can rotate an element in any direction. 2D and 3D.

    Scale: The CSS3 Scale Module allows you to scale an HTML element. With CSS3 Scale, you can make an element larger or smaller.

    Translate: The CSS3 Translate Module allows you to translate an HTML element. With CSS3 Translate, you can move an element in any direction.

    Skew: The CSS3 Skew Module allows you to skew an HTML element. With CSS3 Skew, you can change the direction in degrees of an element.

    The transition-duration: transition duration property defines how long it takes for a property to go from one state to another. The transition-delay transition-delay property defines the time it should take after the property changes before the second transition happens. transition-property: transition-property property defines which CSS properties the transition effect will affect. transition-timing-function: The transition-timing function, allows you to change the pace of a transition by choosing how fast or slow to go. You can slow down part of a transition and speed up other parts. You can even hold each part of the transition for a specific length of time before it transitions to the next state.

    Animations: CSS only changes elements gradually. With CSS transitions, you can change the style of an element by setting rules for how it changes from one state to another. But with CSS animations, you can also tell the browser to change whatever properties you want over time so you can make more complex interactions than simple changes.

    Keyframes: Keyframes are a way to define a series of steps in a CSS animation.

    animation-name: The animation-name property defines the name of the animation. You can create a name for an animation when you make the animation in CSS. You can use this name of your animation instead of the name of your CSS or the @keyframes keyword and the code to make it when you want to show it.

    animation-duration: The animation-duration property defines the length of the animation.

    animation-timing function: The animation-timing-function property defines how the animation will run.

    Html5 Canvas: The HTML5 Canvas element is used to draw graphics, on the fly, via scripting (usually JavaScript).

    SVG: Scalable Vector Graphics (SVG) is an XML-based language for describing vector graphics.

    Drag and drop: The drag and drop feature allows you to drag and drop an element on another element.
